Formula1-API

Welcome to the Formula1-API created in Java with the help of Spring Boot.
The data is provided for free from Vopani and hosted on the Kaggle website. You can check it out here: https://www.kaggle.com/rohanrao/formula-1-world-championship-1950-2020

1 Circuit

Name Type Description
id Long The unique value to identify the Circuit object.
circuitRef String Another way to identify the Circuit, a short handle for the Circuit name.
name String The full name of the Circuit.
location String The city/region in which the Circuit takes place.
country String The country in which the Circuit takes place.
lat double The Circuit’s latitude;
long double The Circuit’s longitude;
alt double The Circuit’s altitude;
url String An URL to the Circuit’s wikipedia page.

1.1 /api/circuits

Request Type Api Endpoint Description
GET /api/circuits Get all circuits

Request

curl --location --request GET 'https://formula1data-api.herokuapp.com/api/circuits'

Response

"_embedded": {
        "circuitList": [
            {
                "id": 853,
                "circuitRef": "albert_park",
