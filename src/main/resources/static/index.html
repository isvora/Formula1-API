Formula1-API


Welcome to the Formula1-API created in Java with the help of Spring Boot.
The data is provided for free from Vopani and hosted on the Kaggle website. You can check it out here: https://www.kaggle.com/rohanrao/formula-1-world-championship-1950-2020

Head over to the docs section to get started

Docs

Here you can find all the API calls you can make with example and responses.

1 Circuits

There are 9 fields in the Circuit object:

Name Type Description
id Long The unique value to identify the Circuit object.
circuitRef String Another way to identify the Circuit, a short handle for the Circuit name.
name String The full name of the Circuit.
location String The city/region in which the Circuit takes place.
country String The country in which the Circuit takes place.
lat double The Circuit latitude;
long double The Circuit longitude;
alt double The Circuit altitude;
url String An URL to the Circuit wikipedia page.

1.1 /api/circuits

GET /api/circuits

curl --location --request GET 'https://formula1-api-demo.herokuapp.com/api/circuits'

1.2 /api/circuits/{id}

In order to retrieve a single Circuit we can pass the id of that circuit

Request Type Api Endpoint
GET /api/circuits/{id}

Request

curl --location --request GET 'https://formula1-api-demo.herokuapp.com/api/circuits/1'

Response

    "id": 1,
